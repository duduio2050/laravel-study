# Start Laravel Portpolio
# 목적
1. 프레임워크를 직접 구현하여 기술 향상.

# 개발 환경
1. PHP 5.7<br>
2. Laravel framwork 7.1.3ㅣ<br>
3. MySQL 5.6

# 개발 도구
1. PHPStorm

# 목표
1. 전반적으로 라라벨을 사용하면서 사용되는 기능들을 모두 구현시켜보는 것.

# 프로젝트 구조
1. app - 모델, 컨트롤러를 담은 객체 폴더<br>
2. Model - 데이터베이스<br>
3. Controller <br>
4. route - 라우팅<br>
5. view - 뷰 파일<br>
